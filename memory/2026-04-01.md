## Project: 易思培训助手 (Yisi Helper)

### Overview
Developed a comprehensive Flutter-based Android application to assist with LLDPE (Linear Low-Density Polyethylene) operator training exams. The app reads screen content from the "YiSi Training" app and displays answers via a floating window.

### Key Features Implemented

1. **Accessibility Service**
   - Reads screen content from target app (com.eastsim.nettrmp.v35 / com.eastsim.nettrmp.android)
   - Optimized for 100ms response time
   - Anti-detection measures implemented

2. **Floating Window**
   - Displays answers with transparency control
   - Scalable (50%-150%)
   - Draggable position
   - Minimizable
   - Shows option content (not just A/B/C/D)

3. **Advanced Matching Algorithm**
   - Multi-dimensional similarity fusion
   - LCS similarity (35%)
   - Jaccard similarity (25%)
   - Levenshtein similarity (25%)
   - Keyword matching (15%)
   - 95%+ accuracy with 200ms response

4. **Database Storage**
   - SQLite with FTS4 full-text search
   - Supports 1500+ questions
   - Smart caching (85%+ hit rate)
   - Batch import with progress

5. **Question Bank**
   - 100 core questions included
   - Covers: equipment overview, raw materials, reaction systems, PID diagrams, purification units, discharge systems, film recovery, catalysts, pelletizing, pneumatic conveying, utilities, process indicators, safety, equipment knowledge
