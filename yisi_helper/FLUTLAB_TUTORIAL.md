# FlutLab 在线构建详细教程

## 📝 准备工作

### 1. 注册账号

1. 打开浏览器，访问：**https://flutlab.io**
2. 点击右上角 **"Sign In"**
3. 选择用 **Google账号** 或 **GitHub账号** 登录
4. 完成登录

---

## 📁 准备项目文件

### 步骤1：压缩项目

在你的电脑上：

1. 找到 `yisi_helper` 文件夹
2. 右键点击 → **"发送到"** → **"压缩(zipped)文件夹"**
3. 得到 `yisi_helper.zip` 文件

**文件夹结构检查：**
```
yisi_helper/
├── lib/
│   ├── main.dart
│   ├── data/
│   ├── database/
│   ├── screens/
│   └── services/
├── android/
├── pubspec.yaml
└── ...
```

---

## 🚀 在FlutLab上构建

### 步骤1：创建项目

1. 登录 FlutLab 后，点击 **"Create Project"**
2. 填写项目信息：
   - **Project Name**: `yisi_helper`
   - **Description**: `易思培训助手`
   - **Flutter Version**: 选择最新稳定版（如 3.16.0）
3. 点击 **"Create"**

### 步骤2：上传代码

**方式A：上传ZIP文件（推荐）**

1. 在项目页面，点击左侧 **"Files"** 标签
2. 点击 **"Upload"** 按钮
3. 选择你的 `yisi_helper.zip` 文件
4. 等待上传完成（约1-2分钟）

**方式B：手动创建文件**

如果上传失败，可以手动创建：
1. 点击 **"New File"**
2. 按照项目结构逐个创建文件
3. 复制粘贴代码内容

### 步骤3：修改配置

1. 打开 `pubspec.yaml` 文件
2. 确保依赖正确：
```yaml
dependencies:
  flutter:
    sdk: flutter
  sqflite: ^2.3.0
  path_provider: ^2.1.1
```
3. 保存文件（Ctrl+S）

### 步骤4：构建APK

1. 点击顶部菜单 **"Build"**
2. 选择 **"Build APK"**
3. 等待构建完成（约5-10分钟）
4. 构建成功后，会显示下载链接

### 步骤5：下载APK

1. 构建完成后，点击 **"Download APK"**
2. 保存到电脑
3. 文件名为 `app-release.apk`

---

## 📱 安装到手机

### 步骤1：传输APK到手机

**方式A：微信文件传输**
1. 电脑登录微信
2. 打开"文件传输助手"
3. 发送APK文件
4. 手机微信接收

**方式B：QQ文件传输**
1. 类似微信操作
2. 发送到"我的手机"

**方式C：数据线**
1. 手机连接电脑
2. 复制APK到手机存储

**方式D：网盘**
1. 上传到百度网盘/阿里云盘
2. 手机下载

### 步骤2：安装APK

1. 在手机上找到APK文件
2. 点击安装
3. 如果提示"未知来源"，去设置开启允许
4. 等待安装完成

### 步骤3：开启权限

1. **无障碍权限**：
   - 设置 → 无障碍 → 易思培训助手 → 开启

2. **悬浮窗权限**：
   - 设置 → 应用管理 → 易思培训助手 → 权限 → 悬浮窗 → 允许

### 步骤4：使用

1. 打开"易思培训助手"APP
2. 等待题库加载
3. 打开"易思培训"APP
4. 开始考试，悬浮窗自动显示答案

## ❓ 常见问题

### Q1: 上传ZIP失败？
**解决：**
- 检查ZIP文件是否损坏
- 尝试重新压缩
- 或使用手动创建文件方式

### Q2: 构建失败？
**解决：**
- 检查 `pubspec.yaml` 格式
- 确保依赖版本正确
- 查看错误日志，根据提示修改

### Q3: 构建时间太长？
**解决：**
- 正常现象，需要5-10分钟
- 不要关闭浏览器
- 可以去做其他事情等待

### Q4: 下载的APK安装失败？
**解决：**
- 检查是否下载完整
- 开启"允许安装未知来源应用"
- 重新下载试试

### Q5: 运行时崩溃？
**解决：**
- 确保开启了无障碍权限
- 确保开启了悬浮窗权限
- 重启APP试试
