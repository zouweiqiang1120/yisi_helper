# 易思培训助手 - 构建指南

## 📱 目标APP信息

**APP名称**: 易思培训  
**开发商**: 北京东方仿真控制技术有限公司  
**支持包名**:
- 新版: `com.eastsim.nettrmp.v35`
- 旧版: `com.eastsim.nettrmp.android`

## 🛠️ 构建步骤

### 1. 环境准备

确保已安装:
- Flutter SDK (>=3.0.0)
- Android Studio
- Android SDK
- JDK

检查环境:
```bash
flutter doctor
```

### 2. 获取项目

```bash
cd yisi_helper
```

### 3. 安装依赖

```bash
flutter pub get
```

### 4. 构建APK

#### 开发调试版
```bash
flutter run
```

#### 发布版APK
```bash
flutter build apk --release
```

构建完成后，APK文件位于:
```
build/app/outputs/flutter-apk/app-release.apk
```

## 📋 安装与配置

### 1. 安装APK
将 `app-release.apk` 安装到Android手机

### 2. 开启无障碍服务
1. 打开 **设置** → **无障碍**
2. 找到 **易思培训助手**
3. 开启服务
4. 确认弹出的权限请求

### 3. 开启悬浮窗权限
1. 打开 **设置** → **应用管理** → **易思培训助手**
2. 点击 **权限**
3. 开启 **悬浮窗** 权限

### 4. 使用
1. 打开 **易思培训助手** APP
2. 点击 **"显示悬浮窗"** 按钮
3. 打开 **易思培训** APP开始考试
4. 悬浮窗会自动显示识别到的题目答案

## 🔧 功能说明

### 题目识别
- 自动读取易思培训APP的屏幕内容
- 使用LCS算法匹配题库中的题目
- 相似度阈值: 0.6 (可调整)

### 答案显示
- 半透明悬浮窗显示在屏幕上方
- 显示题目类型和答案
- 支持拖动调整位置
- 可关闭悬浮窗

### 题库内容
当前包含约 **100道核心题目**，涵盖:
- 装置概况与工艺流程
- 原料与催化剂
- 反应系统
- PID图与仪表
- 精制单元
- 出料与脱气系统
- 膜回收与火炬系统
- 催化剂系统
- 造粒系统
- 风送系统
- 公用工程
- 工艺指标
- 安全与环保
- 设备知识

## ⚠️ 注意事项

1. **无障碍权限必须开启**，否则无法读取屏幕内容
2. **悬浮窗权限必须开启**，否则无法显示答案
3. 确保易思培训APP是 **新版V35** 或 **旧版Android** 版本
4. 首次使用建议先测试几道题目确认匹配正常

## 🐛 故障排除

### 无法识别题目
- 检查无障碍服务是否开启
- 确认易思培训APP版本是否支持
- 检查悬浮窗是否显示

### 答案不匹配
- 可能是题库中没有该题目
- 题目文本可能有变化
- 可调整匹配阈值重新构建

### 悬浮窗不显示
- 检查悬浮窗权限是否开启
- 重启APP后重试
