# 高级匹配算法说明

## 🎯 算法概述

采用**多维度相似度融合算法**，综合4种相似度计算方法，提高题目匹配准确率。

## 📊 相似度计算方法

### 1. LCS相似度 (权重: 35%)
**最长公共子序列 (Longest Common Subsequence)**

```
LCS分数 = (2 × LCS长度) / (文本1长度 + 文本2长度)
```

**优点**: 能识别顺序相似的文本  
**适用**: 题目表述略有变化的情况

### 2. Jaccard相似度 (权重: 25%)
**字符集合的交集/并集**

```
Jaccard分数 = 交集字符数 / 并集字符数
```

**优点**: 不受字符顺序影响  
**适用**: 关键词匹配

### 3. Levenshtein相似度 (权重: 25%)
**编辑距离 (Levenshtein Distance)**

```
Levenshtein分数 = 1 - (编辑距离 / 最大长度)
```

**优点**: 能识别拼写错误和细微差异  
**适用**: 容错匹配

### 4. 关键词匹配度 (权重: 15%)
**提取关键词后的集合匹配**

```
关键词分数 = 共同关键词数 / 总关键词数
```

**优点**: 关注核心概念  
**适用**: 语义匹配

## 🔢 综合分数计算

```
综合分数 = LCS×0.35 + Jaccard×0.25 + Levenshtein×0.25 + 关键词×0.15
```

| 阈值 | 说明 | 置信度 |
|------|------|--------|

| 0.5-0.75 | 低置信度匹配 | 🔴 低 |

- 避免频繁触发匹配

### 3. 内存优化

- 减少内存占用

## 📝 代码使用

```dart
// 创建匹配器
final matcher = AdvancedQuestionMatcher();

// 加载题库
matcher.loadQuestions(questions);

// 匹配题目
final result = matcher.findBestMatch(screenText);

if (result != null) {
  print('答案: ${result.answer}');
  print('置信度: ${result.confidence}');
  print('相似度: ${result.score}');
}

// 获取多个候选
final candidates = matcher.findTopMatches(screenText, topN: 3);
```
