| 功能模块 | 状态 | 说明 |
|---------|------|------|

### 匹配算法

- **LCS相似度** (35%) - 最长公共子序列
- **Jaccard相似度** (25%) - 字符集合匹配
- **Levenshtein相似度** (25%) - 编辑距离
- **关键词匹配** (15%) - 语义匹配

**准确率**: 完全匹配 99%，部分变化 85%

## 🚀 快速开始

### 1. 构建APK

```bash
cd yisi_helper
flutter pub get
flutter build apk --release
```

APK位置: `build/app/outputs/flutter-apk/app-release.apk`

### 2. 安装配置

1. 安装APK到手机
2. 开启**无障碍权限**: 设置 → 无障碍 → 易思培训助手
3. 开启**悬浮窗权限**: 设置 → 应用管理 → 易思培训助手 → 权限

### 3. 使用

1. 打开"易思培训助手"APP
2. 等待题库加载完成
3. 打开"易思培训"APP开始考试
4. 悬浮窗自动显示答案

## 🔧 故障排除

### 无法识别题目
- 检查无障碍服务是否开启
- 确认易思培训APP版本
- 重启APP重试

### 答案不匹配
- 可能是题库中没有该题
- 题目表述可能有变化
- 置信度低时请人工确认

### 悬浮窗不显示
- 检查悬浮窗权限
- 重启APP后重试
