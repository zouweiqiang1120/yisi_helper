# FlutLab 构建 - 快速参考

## 3步走：

### 第1步：准备（2分钟）
```
1. 访问 https://flutlab.io
2. 用Google/GitHub登录
3. 将 yisi_helper 文件夹压缩成 zip
```

### 第2步：上传（3分钟）
```
1. 点击 "Create Project"
2. 项目名：yisi_helper
3. 点击 "Files" → "Upload"
4. 选择 zip 文件上传
```

### 第3步：构建（10分钟）
```
1. 点击 "Build" → "Build APK"
2. 等待构建完成
3. 点击 "Download APK"
4. 传到手机安装
```

---

## 安装后必做：

```
1. 设置 → 无障碍 → 易思培训助手 → 开启
2. 设置 → 应用管理 → 易思培训助手 → 权限 → 悬浮窗 → 允许
3. 打开易思培训APP开始考试
```

---

## 遇到问题？

| 问题 | 解决 |
|------|------|
| 上传失败 | 重新压缩zip |
| 构建失败 | 检查pubspec.yaml |
| 安装失败 | 开启"允许未知来源" |
| 不显示答案 | 检查权限是否开启 |

---
