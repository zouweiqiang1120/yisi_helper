# 模块2：技术分析

## 2.1 K线图基础

### 什么是K线
K线图（Candlestick Chart）起源于日本，是最常用的股票价格走势图。每根K线代表一个时间周期（日、周、月等）的价格变化。

### K线构成
```
    上影线
       │
    ┌──┴──┬ ← 最高价
    │     │
    │ 实体 │ ← 开盘价/收盘价
    │     │
    └──┬──┘ ← 最低价
       │
    下影线
```

- **实体**：开盘价与收盘价之间的矩形
- **上影线**：实体上方细线，最高价到实体边缘
- **下影线**：实体下方细线，最低价到实体边缘

### K线类型

#### 阳线（上涨）
- 收盘价 > 开盘价
- 通常用红色或空心表示
- 实体下端=开盘价，实体上端=收盘价

#### 阴线（下跌）
- 收盘价 < 开盘价
- 通常用绿色或实心表示
- 实体上端=开盘价，实体下端=收盘价

#### 特殊情况
- **十字星**：开盘价≈收盘价，实体极小
- **一字板**：开盘价=收盘价=最高价=最低价（涨停/跌停）
- **T字板**：开盘价=收盘价=最高价，有下影线

### 常见K线形态

#### 单根K线
| 形态 | 特征 | 含义 |
|------|------|------|
| 大阳线 | 实体很长，涨幅大 | 强势上涨 |
| 大阴线 | 实体很长，跌幅大 | 强势下跌 |
| 锤子线 | 下影线长，实体小 | 可能见底 |
| 倒锤子 | 上影线长，实体小 | 可能见顶 |
| 十字星 | 实体极小 | 多空平衡 |

#### 多根K线组合
| 形态 | 特征 | 含义 |
|------|------|------|
| 早晨之星 | 阴→十字→阳 | 底部反转信号 |
| 黄昏之星 | 阳→十字→阴 | 顶部反转信号 |
| 红三兵 | 连续三根阳线 | 强势上涨 |
| 三只乌鸦 | 连续三根阴线 | 强势下跌 |
| 吞没形态 | 后一根完全包含前一根 | 趋势反转 |

## 2.2 常用技术指标

### 移动平均线（MA）

#### 概念
移动平均线是将某段时间内的收盘价相加后除以天数，得到的平均值连线。

#### 常见周期
- **MA5**：5日均线（周线）
- **MA10**：10日均线（半月线）
- **MA20**：20日均线（月线）
- **MA60**：60日均线（季线）
- **MA120/250**：年线

#### 应用法则
- **金叉**：短期均线上穿长期均线，买入信号
- **死叉**：短期均线下穿长期均线，卖出信号
- **多头排列**：短期>中期>长期均线，上涨趋势
- **空头排列**：短期<中期<长期均线，下跌趋势
- **均线支撑/压力**：价格回调到均线附近获得支撑或遇到压力

### MACD（指数平滑异同移动平均线）

#### 构成
- **DIF线**：12日EMA - 26日EMA
- **DEA线**：DIF的9日EMA
- **MACD柱**：(DIF - DEA) × 2

#### 应用法则
- **金叉**：DIF上穿DEA，买入信号
- **死叉**：DIF下穿DEA，卖出信号
- **零轴上方**：多头市场
- **零轴下方**：空头市场
- **顶背离**：价格新高，MACD未新高，可能见顶
- **底背离**：价格新低，MACD未新低，可能见底

### KDJ（随机指标）

#### 构成
- **K线**：快速确认线
- **D线**：慢速主干线
- **J线**：方向敏感线（J = 3K - 2D）

#### 应用法则
- **超买区**：K、D、J > 80，考虑卖出
- **超卖区**：K、D、J < 20，考虑买入
- **金叉**：K上穿D，买入信号
- **死叉**：K下穿D，卖出信号

### RSI（相对强弱指标）

#### 概念
衡量买卖双方力量对比，取值范围0-100。

#### 应用法则
- **RSI > 70**：超买，可能回调
- **RSI < 30**：超卖，可能反弹
- **顶背离/底背离**：与MACD类似

### 布林带（BOLL）

#### 构成
- **中轨**：20日均线
- **上轨**：中轨 + 2倍标准差
- **下轨**：中轨 - 2倍标准差

#### 应用法则
- **开口**：波动加大，可能有大行情
- **收口**：波动减小，可能即将变盘
- **触及上轨**：可能超买
- **触及下轨**：可能超卖
- **中轨支撑/压力**：价格围绕中轨波动

### 成交量指标

#### 量价关系
| 情况 | 特征 | 含义 |
|------|------|------|
| 价涨量增 | 价格上涨，成交量放大 | 上涨趋势确认 |
| 价涨量缩 | 价格上涨，成交量缩小 | 上涨动力不足 |
| 价跌量增 | 价格下跌，成交量放大 | 恐慌抛售 |
| 价跌量缩 | 价格下跌，成交量缩小 | 抛压减轻 |

## 2.3 形态分析

### 反转形态

#### 头肩顶/底
```
头肩顶：          头肩底：
    ╱╲              ╲╱
   ╱  ╲            ╱  ╲
  ╱    ╲╱╲        ╲╱    ╲
 ╱          ╲            ╲
╱            ╲            ╲
```
- 头肩顶：顶部反转信号
- 头肩底：底部反转信号
- 颈线突破确认

#### 双顶/双底（M头/W底）
```
M头：      W底：
╱╲        ╲╱
  ╲╱      ╱  ╲
```
- 两个相近的高点/低点
- 颈线突破确认

#### 圆弧顶/底
- 缓慢形成的顶部/底部
- 成交量逐渐萎缩后放大

### 持续形态

#### 三角形
- **上升三角形**：水平上轨，上升下轨，看涨
- **下降三角形**：水平下轨，下降上轨，看跌
- **对称三角形**：收敛形态，突破方向不确定

#### 旗形/楔形
- **旗形**：短暂反向倾斜的平行四边形
- **楔形**：收敛的倾斜通道
- 通常是中继形态，趋势延续

#### 矩形
- 价格在水平支撑和压力位之间震荡
- 突破后量度涨幅/跌幅

## 2.4 支撑与阻力

### 支撑位
价格下跌时可能停止下跌并反弹的价格水平。

**常见支撑**：
- 前期低点
- 整数关口
- 均线位置
- 趋势线
- 黄金分割位

### 阻力位
价格上涨时可能停止上涨并回落的价格水平。

**常见阻力**：
- 前期高点
- 整数关口
- 均线位置
- 趋势线
- 密集成交区

### 支撑阻力转换
- 支撑被跌破后变成阻力
- 阻力被突破后变成支撑

## 2.5 趋势线

### 上升趋势线
连接两个或多个低点，向上倾斜。价格回调到趋势线附近是买入机会。

### 下降趋势线
连接两个或多个高点，向下倾斜。价格反弹到趋势线附近是卖出机会。

### 通道线
与趋势线平行的另一条线，形成价格运行的通道。
