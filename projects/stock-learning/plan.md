# 股票学习计划

## 目标
- 掌握股票投资基础知识
- 学会技术分析方法
- 学会基本面分析方法
- 能够辅助分析具体股票

## 学习模块

### 模块1：基础知识
- [ ] 股票市场基础概念（A股、港股、美股等）
- [ ] 股票交易规则（T+1、涨跌停、交易时间等）
- [ ] 常用术语（PE、PB、ROE、市值等）
- [ ] 投资流派介绍（价值投资、成长投资、趋势投资等）

### 模块2：技术分析
- [ ] K线图基础
- [ ] 常见技术指标（MA、MACD、KDJ、RSI、布林带等）
- [ ] 形态分析（头肩顶/底、双顶/底、三角形等）
- [ ] 成交量分析
- [ ] 支撑与阻力位

### 模块3：基本面分析
- [ ] 财务报表分析（资产负债表、利润表、现金流量表）
- [ ] 关键财务指标（盈利能力、偿债能力、运营效率）
- [ ] 行业分析
- [ ] 估值方法（PE、PB、DCF等）
- [ ] 宏观经济影响

### 模块4：实践应用
- [ ] 搭建股票分析框架
- [ ] 获取股票数据的方法
- [ ] 实际案例分析

- 开始时间：2026-04-02
