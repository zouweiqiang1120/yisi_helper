# 模块3：基本面分析

## 3.1 财务报表基础

上市公司每年发布四份定期报告：
- **一季报**：4月发布
- **半年报**：8月发布
- **三季报**：10月发布
- **年报**：次年4月发布（最重要）

### 三大财务报表

#### 资产负债表（某一时点的财务状况）
反映公司在特定日期的资产、负债和股东权益。

**基本公式**：资产 = 负债 + 股东权益

| 资产 | 负债和股东权益 |
|------|----------------|
| 流动资产 | 流动负债 |
| - 货币资金 | - 短期借款 |
| - 应收账款 | - 应付账款 |
| - 存货 | 非流动负债 |
| 非流动资产 | - 长期借款 |
| - 固定资产 | 股东权益 |
| - 无形资产 | - 股本 |
| | - 资本公积 |
| | - 未分配利润 |

#### 利润表（一段时间的经营成果）
反映公司在一定期间内的收入、成本和利润。

**核心公式**：净利润 = 营业收入 - 营业成本 - 费用 - 税金

主要项目：
- **营业收入**：卖产品/服务收到的钱
- **营业成本**：生产产品的直接成本
- **毛利** = 营业收入 - 营业成本
- **毛利率** = 毛利 / 营业收入
- **营业利润**：核心业务的利润
- **净利润**：最终赚到的钱
- **归母净利润**：归属于上市公司股东的净利润

#### 现金流量表（一段时间的现金流动）
反映公司现金的流入和流出。

**三大活动**：
1. **经营活动现金流**：主营业务产生的现金
   - 销售商品收到的现金
   - 购买商品支付的现金
   - 理想状态：持续为正且大于净利润

2. **投资活动现金流**：投资产生的现金
   - 购买固定资产
   - 对外投资
   - 成长期通常为负（扩张投资）

3. **筹资活动现金流**：融资产生的现金
   - 借款
   - 发行股票
   - 分红
   - 还债

**重要指标**：
- **自由现金流** = 经营现金流 - 资本支出
- 正值表示公司能自我造血

## 3.2 关键财务指标

### 盈利能力指标

| 指标 | 公式 | 含义 |
|------|------|------|
| **毛利率** | (营收-成本)/营收 | 产品竞争力，越高越好 |
| **净利率** | 净利润/营业收入 | 最终盈利能力 |
| **ROE** | 净利润/净资产 | 股东投资回报率，核心指标 |
| **ROA** | 净利润/总资产 | 资产利用效率 |

**ROE分析（杜邦分析法）**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
    = (净利润/营收) × (营收/总资产) × (总资产/净资产)
```

- **高净利率**：茅台模式（品牌溢价）
- **高周转率**：沃尔玛模式（薄利多销）
- **高杠杆**：银行模式（负债经营）

### 成长能力指标

| 指标 | 公式 | 含义 |
|------|------|------|
| **营收增长率** | (本期营收-上期营收)/上期营收 | 业务扩张速度 |
| **净利润增长率** | (本期净利-上期净利)/上期净利 | 盈利增长速度 |
| **扣非净利润增长率** | 扣除非经常性损益后的增长 | 真实经营增长 |

### 偿债能力指标

| 指标 | 公式 | 含义 |
|------|------|------|
| **资产负债率** | 总负债/总资产 | 财务杠杆，一般<70% |
| **流动比率** | 流动资产/流动负债 | 短期偿债能力，>2较好 |
| **速动比率** | (流动资产-存货)/流动负债 | 更严格的短期偿债 |

### 运营效率指标

| 指标 | 公式 | 含义 |
|------|------|------|
| **应收账款周转天数** | 365/(营收/平均应收) | 回款速度，越短越好 |
| **存货周转天数** | 365/(成本/平均存货) | 库存管理效率 |
| **总资产周转率** | 营收/总资产 | 资产利用效率 |

## 3.3 估值方法

### 相对估值法

#### 市盈率（PE）
```
PE = 股价 / 每股收益(EPS)
   = 总市值 / 净利润
```

- **静态PE**：用上年净利润
- **动态PE**：用预测净利润
- **TTM PE**：用最近四个季度净利润

**应用**：
- 同行业比较
- 与历史PE比较
- 成长股PE通常较高
- 银行股PE通常较低

#### 市净率（PB）
```
PB = 股价 / 每股净资产
   = 总市值 / 净资产
```

- 适用于重资产行业（银行、地产）
- PB < 1 可能低估（也可能有问题）

#### 市销率（PS）
```
PS = 总市值 / 营业收入
```

- 适用于亏损但高增长的互联网公司

#### PEG比率
```
PEG = PE / 盈利增长率
```

- PEG < 1：可能被低估
- PEG > 2：可能被高估
- 适用于成长股

### 绝对估值法

#### 现金流折现（DCF）
```
企业价值 = Σ(未来自由现金流 / (1+折现率)^年数)
```

**步骤**：
1. 预测未来5-10年自由现金流
2. 确定合适的折现率（WACC）
3. 计算终值
4. 折现求和

**难点**：预测难度大，参数敏感

#### 股利折现模型（DDM）
适用于稳定分红的公司：
```
股价 = 预期股利 / (要求回报率 - 股利增长率)
```

## 3.4 行业分析

### 行业生命周期

| 阶段 | 特征 | 投资策略 |
|------|------|----------|
| **导入期** | 技术不成熟，市场小 | 高风险高回报 |
| **成长期** | 快速增长，竞争加剧 | 最佳投资期 |
| **成熟期** | 增长放缓，龙头集中 | 稳健收益 |
| **衰退期** | 需求下降 | 避免投资 |

### 行业分析框架（波特五力）

1. **现有竞争者**：行业竞争激烈程度
2. **潜在进入者**：进入壁垒高低
3. **替代品威胁**：被新技术替代的风险
4. **供应商议价能力**：上游控制力
5. **客户议价能力**：下游控制力

### 好行业的特征
- 需求持续增长
- 进入壁垒高
- 竞争格局稳定
- 定价能力强
- 轻资产运营

## 3.5 公司分析框架

### 护城河分析

| 类型 | 说明 | 例子 |
|------|------|------|
| **品牌护城河** | 强大的品牌认知 | 茅台、可口可乐 |
| **成本优势** | 比别人成本低 | 格力电器、福耀玻璃 |
| **网络效应** | 用户越多价值越大 | 腾讯、阿里巴巴 |
| **转换成本** | 客户难以切换 | 办公软件、数据库 |
| **规模效应** | 越大越有效率 | 快递、零售 |

### 管理层分析
- 诚信记录
- 专业能力
- 股东利益一致性
- 战略眼光

### 分析清单

#### 定性分析
- [ ] 商业模式是否简单易懂？
- [ ] 是否有持续竞争优势？
- [ ] 行业前景如何？
- [ ] 管理层是否优秀？
- [ ] 公司文化是否健康？

#### 定量分析
- [ ] ROE是否持续>15%？
- [ ] 毛利率是否稳定或提升？
- [ ] 营收和利润是否持续增长？
- [ ] 负债率是否合理？
- [ ] 现金流是否健康？
- [ ] 估值是否合理？

## 3.6 宏观经济影响

### 关注指标
- **GDP增速**：经济整体状况
- **CPI/PPI**：通胀水平
- **利率**：影响资金成本和估值
- **汇率**：影响进出口企业
- **M2增速**：货币供应量

### 经济周期与行业
| 周期 | 表现行业 |
|------|----------|
| 复苏期 | 金融、地产、建材 |
| 扩张期 | 科技、消费、工业 |
| 滞胀期 | 能源、原材料、公用事业 |
| 衰退期 | 医药、必选消费、公用事业 |
